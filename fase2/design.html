Fundamentos de HTML

Que conocemos como HTML

El HTML es el lenguaje de marcado el cual es el estándar actual para la creación de las paginas web, especificando que es HTML, este tipo de lenguaje por sus silabas significa lenguaje de marcado de hipertexto o HTML, este código describe la estructura de la pagina web, y con esta estructura el navegador reconocer y puede mostrar el contenido que sea fácil de leer para las personas, de manera simple el navegador toma las etiquetas de encabezados, párrafos entre otras y las traduce para que nosotros como usuarios las podamos entender fácilmente.

Estructura básica de un archivo HTML

La estructura básica de un archivo HTML puede ya mostrar contenido en pantalla y esta se organiza de la siguiente manera:

<!DOCTYPE html>
<html>
<head>
<title>Titulo</title>
</head>
<body>
<h1>Titulo contenido</h1>
<p>Párrafo</p>
</body>
</html>

Las etiquetas mostradas anteriormente mostradas tienen una explicación y un detalle del porqué, de cada una de ellas, a continuación, se detallan las mismas:

  • <!DOCTYPE html>: Etiqueta especifica si es un documento con tecnología HTML5
  • <html>: Etiqueta el elemento raíz de toda página web HTML
  • <head>: Etiqueta que contiene toda la metainformación de la página HMTL
  • <title>: Etiqueta que especifica el titulo de la página web (el nombre que aparece en el navegador)
  • <body>: Etiqueta que define el contenido del documento es el contenedor donde se muestran todos los objetos visuales de la pagina
  • <h1>: Etiqueta que especifica el encabezado principal de la pagina
  • <p>: Etiqueta que especifica un párrafo en la pagina

Elementos de un HTML

La estructura de un elemento HTML se define mediante una etiqueta de inicio, luego un contenido y después se define una etiqueta de finalización.

<nombre de la etiqueta> El contenido va aquí... </nombre de la etiqueta>

Se conoce como elemento HTML todo el contenido de la etiqueta de inicio hasta la etiqueta final\

<h1> Mi primer encabezado </h1>

<p> Mi primer párrafo. </p>

Hay etiquetas que tienen que solo tiene una etiqueta de inicio y final y hay otras etiquetas que solo tiene etiqueta de inicio.

Como funcionan los navegadores web

El propósito de los navegadores web como lo son (Chrome, Edge, Firefox, Safari) es leer cada uno de los documentos HTML y mostrarlos de manera indicada. Los navegadores no muestran las etiquetas HTML, pero este las usa para determinar como se muestra el contenido.

Historia resumida de las versiones de HTML

  • 1989: Tim Berners-Lee invento el www o el internet.
  • 1991: Tim Berners-Lee invento la estructura HTML.
  • 1993: David Raggett realizo la estructuración del HTML +
  • 1995: Se establece el grupo de trabajo HTML en el cual se avanza y define la versión HTML 2.0
  • 1997: Recomendaciones W3C para la estructuración de HTML 3.2
  • 1999: Recomendaciones W3C para la estructuración de HTML 4.01
  • 2000: Recomendaciones W3C para la estructuración de XHTML 1.0
  • 2008: Primer borrador público de WHATWG he incorporación en HTML 5
  • 2012: Estructuración de WHATWG en el estándar de vida en HTML 5
  • 2014: Recomendaciones W3C para la estructuración de HTML 5
  • 2016: Recomendaciones W3C para la versión candidata de HTML 5.1
  • 2017: Recomendaciones W3C para la estructuración de HTML 5.1 segunda edición
  • 2017: Recomendaciones W3C para la estructuración de HTML 5.2

Interacción Humano Ordenador

Interacción hombre-computadora (HCI)

Se conoce como a la interacción de humano – computadora es uno de los campos de estudios multidisciplinarios que permite la creación y el diseño de tecnologías informáticas y por lo general la interacción entre usuario y los computadores. En un principio se utilizó para la relación del humano con solo las computadoras, pero actualmente abarca casi abarca todos los avances tecnológicos de la humanidad.

El acenso de la interacción hombre-computadora

Esta interacción se comenzó a estudiar en la década de los 80’ con la llegada de la computación personal con computadores como el Apple Macintosh y el PC de IB entre otros, cuando estos equipos llegaron al consumidor en general estas máquinas ya poseían software bastante sofisticados como lo son los procesadores de texto, videojuegos, sistemas de contabilidad entre otros, esto trajo como consecuencia que cada vez las computadoras fueras mas accesibles y ya no solo eran utilizadas en entornos científicos, es por eso que se presento la necesidad de crear una interacción entre el humano y la computadora que fuera fácil y eficiente.

Es por eso por lo que el HCI es el objetivo de una intensa investigación académica dada su importancia, las personas que estudiaron y trabajaron en este tipo de interacción lo vieron como una manera para popularizar la idea de que el dialogo de una persona con una computadora debía ser similar al que tiene dos personas, en las primeras investigaciones del HCI las personas intentaron mejorar la usabilidad e las computadoras de escritorio haciéndolas más fáciles de usar. Luego dado el avance tecnológico del internet y teléfonos inteligentes se ve la necesidad de investigar mas en el uso de estas interfases no solo en una computadora si no en internet a través de dispositivos móviles y IOT.

El valor de la interfaz usuario en la interacción humano maquina

El estudio de estas interacciones humano maquina se derivan en diferentes áreas de estudio como lo son el diseño centrado en el usuario (UCD), el diseño de interfaz de usuario (UI) y la experiencia que este usuario obtiene (UX), es por eso por lo que podemos tomar al HCI como un antecesor al diseño de UX.

Por otro lado, aunque sean muy similares este tipo de diseño de interfaces ya que uno es precursor del otro hay algunas diferencias para tener en cuenta, cuando hablamos de la interacción humano maquina se suele hablar de enfoques mas académicos, relacionados con la investigación científica por otro lado los diseñadores de UX se suele centrar en la industria y participan en la creación de productos y servicios reales como pueden ser aplicaciones móviles o Páginas web.
